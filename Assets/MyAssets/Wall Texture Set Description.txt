Instructions:

Create a 3d object (plane, cube, etc.), Scale it as you want, and drag and drop material on it.
You can use these textures as you like.
There are 3 textures on each material, AlbedoTransparency, MetallicSmoothness and Normal.
You can scale material with tiling option to scale texture as you want for performance optimization.
Adjust smoothness if you want.
